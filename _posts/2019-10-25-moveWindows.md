# 問題

今天遇到一個問題:
電腦上的應用程式不管怎麼點都點不開
一開始一直以為是win10的一些bug
後來發現原來是應用程式跑出螢幕外了


# 解決

利用一些移動視窗的快捷鍵將程式從螢幕外拉回來
便成功解決了這個問題^^

	1. win + Tab 或 Alt + Tab : 選取要移動的程式
	2. win + 上下左右 或 win + shift + 上下左右 : 移動視窗
